# Auditoria de Segurança Interna - Botium Toys (NIST CSF)

## 📌 Cenário do Projeto
Este projeto faz parte do **Certificado Profissional de Cibersegurança do Google**. O objetivo foi realizar uma auditoria interna na empresa fictícia "Botium Toys" para avaliar sua postura de segurança e conformidade com normas internacionais.

## 🛠️ O que foi feito:
- Análise de riscos e inventário de ativos.
- Verificação de controles administrativos, técnicos e físicos.
- Avaliação de conformidade com **PCI DSS**, **GDPR** e **SOC**.
- Elaboração de recomendações estratégicas para a gerência de TI.

## 📊 Principais Controles Avaliados
| Controle | Status | Observação |
| :--- | :---: | :--- |
| Firewall | ✅ | Implementado e monitorado. |
| Criptografia | ❌ | Ausente (Risco alto para conformidade). |
| Privilégio Mínimo | ❌ | Todos os funcionários possuem acesso total. |
| IDS | ❌ | Não instalado. |

## 🚀 Conclusões
A empresa apresenta um score de risco de **8/10**. Minhas recomendações focaram na implementação imediata de criptografia para atender ao PCI DSS e na criação de um Plano de Recuperação de Desastres.
